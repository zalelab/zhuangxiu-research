# 装修签合同注意事项

## 一、合同主体信息

- [ ] ⚠️ **确认合同上的公司名称与营业执照完全一致**（不是个人名字），记下营业执照编号——出纠纷时这是你起诉的依据，名字不对连告谁都找不到
- [ ] 记录项目经理/工长姓名、联系方式，确认业主信息和房屋地址无误

---

## 二、工程报价与费用

- [ ] 报价单逐项核对，确认包含所有项目（不留空白项），包括管理费、税费、上楼费、垃圾清运费等隐性费用
- [ ] 明确标注「增项」规则：增项需业主书面签字确认后才可施工，约定增项总额上限（建议不超过合同总价的 **5%**）。⚠️ 施工中工人常以"现场发现问题"为由推荐加购/升级，**未经业主书面确认的增项费用有权拒付**
- [ ] 确认付款方式和比例，建议分期付款：
  - 开工付 **30%**
  - 水电完工付 **30%**
  - 油漆/泥工完成付 **30%**
  - 竣工验收合格付 **10%** 尾款
- [ ] ⚠️ **设计免费陷阱**：装修公司宣称免费设计，但只提供平面图，效果图和施工图需额外收费（3000-5000元），提前问清楚
- [ ] ⚠️ **定金与订金**：定金不可退，订金可退！装修公司常要求交"定金"锁定客户，注意区分
- [ ] ⚠️ **水电增项重灾区**：常见套路有按线头计费、三线单管重复计费，**必须要求水电包死价合同**
- [ ] ⚠️ **签约前必须确定所有主材**，未确定材料就签约，后期加价空间达30-50%
- [ ] 要求装修公司提供**清单式报价**，参考格式如下：

| 项目                 | 单价     | 数量    | 工艺标准          |
| -------------------- | -------- | ------- | ----------------- |
| 墙面乳胶漆           | 65元/㎡  | 60.59㎡ | 腻子两遍+灯照打磨 |
| 地砖铺贴             | 240元/㎡ | 27.34㎡ | 空鼓率≤10%       |
| （所有项目逐项列明） | ...      | ...     | ...               |

### ⚠️ 常见报价套路对照表

| 套路类型             | 常见话术          | 真实情况                               | 防坑对策                                           |
| -------------------- | ----------------- | -------------------------------------- | -------------------------------------------------- |
| **限制性条款** | "橱柜不限米数"    | 限3米地柜+1米吊柜，超量2000元/米加价   | 合同写明具体米数限制及超量单价                     |
| **低价诱导**   | "全屋墙布只要3万" | 报价仅算80㎡，实际需300㎡，总价飙到6万 | 要求按实际用量报价，写进合同                       |
| **隐藏收费**   | "垃圾清运免费"    | 单次清运2000元，全屋清运可超万元       | 所有费用（含清运、搬运、功能柜加价）逐项写入报价单 |
| **材料升级**   | "主材任选"        | 可选范围多为滞销款，升级加价30-50%     | 签约前确定所有主材品牌型号，写入合同               |
| **全屋挂网**   | "必须全屋挂网防开裂" | 全屋挂网收费4000-5000元，实际只需局部挂网 | 只在**新旧墙交接处、水电开槽回填处、石膏板接缝处**局部挂网即可，拒绝全屋挂网 |

## 三、材料相关

- [ ] 合同中明确列出所有主材、辅材的 **品牌、型号、规格、数量**（只承诺品牌不明确型号是大坑，如"九牧"马桶有500-10000元不等的产品，价差巨大）
- [ ] 标注材料是甲供（业主买）还是乙供（装修公司买）
- [ ] 乙供材料进场时业主有权验收，不合格可拒收
- [ ] 约定材料替换需书面告知并经业主同意，**施工中材料包装留存**，每次进场材料与合同清单逐一核对
- [ ] 剩余材料的退换责任归属要写清楚
- [ ] ⚠️ **环保等级必须明确**：ENF级与E1级成本差3倍以上，合同中必须写明板材环保等级（建议要求 **ENF级或E0级**）
- [ ] ⚠️ **渠道专供材料质量差**：装修公司宣称的"进口材料"或"专供产品"多为小厂代工贴牌，质量低于市售款
  - 案例：某公司89元/㎡的强化地板，市场同品牌售价109元/㎡，却向业主报价150元/㎡
  - **要求提前提供完整材料表（含品牌/型号/执行标准），只选电商/建材市场�
